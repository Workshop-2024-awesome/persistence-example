# ❓ What this is about

An example repository for our workshop about persistence and databases in 2024. 

# 💡 How to start

## ⛵️ Required ports
- `5432` for postgres
- `6379` for Redis
- `8888` for pgAdmin

## ⚙️ Required tooling
- Docker (https://docs.docker.com/engine/install/)

## 💻 Starting commands for local env

```bash
# Start infrastructure
> docker compose up -d
# Alternatively (with a dash -)
> docker-compose up -d
```
